# TDKit
Terminal Development Kit

## Project Information
- **Name:** TDKit
- **Author:** nyu3321225
- **GitHub:** https://github.com/nyu3321225
- **Version:** v1.0.0-beta
- **Description:** A collection of tools for terminal development and management.

## Features
- Environment check
- Tool installation
- Storage usage display
- IP address display
- Project skeleton creation

## Installation
- Clone this repository using git:
  ```bash
  git clone https://github.com/nyu3321225/tdkit
  ```
- Or download the tarball and extract it, then run the install script:
  ```bash
  wget https://github.com/nyu3321225/tdkit/archive/v1.0.0-beta.tar.gz
  tar -xzf v1.0.0-beta.tar.gz
  cd tdkit
  bash install.sh
  ```

## Usage
- Run `tdkit` to start the application.
- Use `-e` for environment check, `-i` for tool installation.
